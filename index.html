Launch your bots to the moon!

Cord.js is a simple and unopinionated Discord bot framework.

Get Started
import createClient from '@cordjs/client'
import Gateway from '@cordjs/gateway'

const client = createClient([
  [Gateway, {
      token: process.env.TOKEN,
      client: {
        intents: ['GUILDS', 'GUILD_MESSAGES'],
      },
  }],
])

client.gateway.messageCreate(context => {
  const [message] = context.data

  if (message.content === '!hello-world') {
    return message.reply('Hello world!')
  }
})

client.start()

Easy to Use

Easily get a Discord bot up and running, using the Discord gateway and/or interactions.

Unopinionated

Cord.js is unopinionated, meaning you can structure your bot in any way you want.

Powered using Middleware

Middleware is a powerful way to listen to events, and handle errors.
